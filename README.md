# Opero WPP Use Cases

55 real-world examples of what you can build with [Opero WPP](https://wpp.opero.so) — the WhatsApp API built for AI agents.

Each use case includes a business problem, solution architecture, and working code you can copy-paste. From a simple auto-reply bot to a full AI-powered sales pipeline.

## Quick Start

1. Sign up at [wpp.opero.so](https://wpp.opero.so)
2. Get your API key
3. Pick a use case below and follow the code

## How Each Use Case Works

Every use case follows the same pattern:

```
Incoming WhatsApp Message
    ↓
Your Webhook Server receives it
    ↓
AI classifies / generates response
    ↓
Opero WPP API sends reply + tags with metadata
    ↓
Conversation state tracked via metadata
```

## Tech Stack

Most examples use:
- **Node.js** (Express) for the webhook server
- **Anthropic Claude** or **OpenAI GPT** for AI
- **Opero WPP API** for WhatsApp
- **Metadata** for state management (no external DB needed for simple cases)
